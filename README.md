# Mentoria API


## Padrão de commits
- 🪧: primeiro commit
- ⚙️: configuração
- 🗂️: (rename / move) [arquivo / pasta]
- 🔥: remoção de código / arquivo / pasta
- 📦: dependencies / packages
- 🧪: testes
- 👀: a fazer / a ver
- 🐛: correção de bug
- ✔️: validators
- 📝: models / views
- 🧰: controller / middleware
- 🧩: modulo / feature
- 🧾: exports / imports
- 📑: documentação
- 🔨: html / css
- 🪢: apenas código
